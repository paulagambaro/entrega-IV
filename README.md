# Portfolio de Paula Gâmbaro

## Descrição
Este é o projeto final desenvolvido como parte das Experiências Práticas da disciplina. O portfolio é uma aplicação web que reúne minhas informações pessoais, projetos realizados e formas de contato, apresentando **layout responsivo, interatividade e boas práticas de acessibilidade**.

O projeto simula uma aplicação real e foi desenvolvido para demonstrar habilidades em HTML5, CSS3 e JavaScript, seguindo as exigências de acessibilidade WCAG 2.1 Nível AA.

---

## Tecnologias Utilizadas
- **HTML5**: Estrutura semântica da página  
- **CSS3**: Estilização, layout responsivo com Flexbox e Grid, sistema de cores e tipografia  
- **JavaScript (ES6+)**: Manipulação do DOM, Single Page Application (SPA) simples, validação de formulários, interatividade de cards e modais  

---

## Funcionalidades Implementadas
- Layout responsivo com **CSS Grid e Flexbox**  
- Sistema de cores e tipografia consistente  
- **SPA simples**: navegação sem recarregar a página  
- Cards responsivos para exibição de projetos  
- Botões com estados visuais (hover, focus, active, disabled)  
- Formulário de contato com validação visual e feedback  
- Modais e toasts para mensagens de alerta  
- Sistema de tags e badges para categorização de projetos  
- Navegação acessível por teclado e semântica adequada  
- Contraste de cores adequado para facilitar leitura

## Como Executar
1. Clone ou baixe o repositório  
2. Abra o arquivo `index.html` em um navegador moderno (Chrome, Edge, Firefox)  
3. Navegue pelos menus para acessar informações, projetos e contato  

---

## Acessibilidade
- Navegação por teclado em todos os componentes  
- Estrutura semântica adequada (`header`, `main`, `section`, `footer`)  
- Contraste mínimo de 4.5:1 para textos  
- Modos de visualização alternativos planejados para alto contraste e leitura de tela
